# Flag Quiz

## Описание
Приложение для проверки знаний флагов стран мира. Пользователь видит флаг и выбирает страну из 4 вариантов.

## Установка
1. Клонировать репозиторий
2. Открыть в Android Studio
3. Запустить на устройстве с Android 6.0 (API 23) или выше

## Использование
- На экране отображается флаг
- Нажми на одну из 4 кнопок с названиями стран
- При правильном ответе — зелёное сообщение
- При неправильном — красное сообщение и анимация флага
- После 10 вопросов показывается результат
